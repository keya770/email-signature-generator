How to add your signature

Outlook Desktop: Click Copy for Outlook → File → Options → Mail → Signatures → New → paste (Ctrl+V) inside the editor.

Best for images: Use Download .htm → open in Chrome/Edge → Ctrl+A → Ctrl+C → paste into Outlook. Allow external images when prompted.

Gmail: Copy for Outlook works in Gmail too → Settings → See all settings → General → Signature → paste.

Important: Use Copy for Outlook or Download .htm — not raw HTML paste. In Outlook, right-click signature → Download Pictures if logos don't show.
